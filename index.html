HOWTO по заданию

Выполнил 2 варианта задания v1 и v2.

  • В версии v1 программа запросит ваши Trello API key & Trello API token, а также КОРОТКИЙ ID вашей доски, прошу обратить внимание что именно короткий, длинный программа получит сама
  • В Версии v2 вам необходимо, перед запуском программы, вставить ваши Trello API key & Trello API token & КОРОТКИЙ ID вашей доски

    • Получить короткий ID вы можете, например, из адреса странички необходимой доски. https://trello.com/b/ТУТ_КОРОТКИЙ_ID


    Варианты запуска программы:
  • без параметров - программа выводит на экран колонки и задачи в них
  • с параметром "create имя колонка" - создаст задачу _имя_ в _колонка_
  • с параметром "move name имя колонка" - переместит задачу _имя_ в _колонка_, если задач несколько, предлжит выбор
  • с параметром "listadd колонка" - создаст колонку _колонка_
